    • Booking Simulator

    This project is a web application designed for Cozy Apartments, a service for booking and managing apartment rentals. The application leverages several programming languages and technologies to enhance functionality and interactivity. It utilizes **HTML** for structuring content, **JavaScript** for handling user interactions and form submissions, and **CSS** for styling the layout and design. The JavaScript code also fetches real-time currency exchange rates from an external API , displaying both official and unofficial (blue market) dollar rates, which update every few seconds. Overall, this web application provides an interactive and user-friendly platform for booking Cozy Apartments, making reservations, and staying informed about currency exchange rates, enhancing the user experience for potential guests.
